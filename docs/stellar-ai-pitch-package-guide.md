## WHAT NOT TO REVEAL

### Protect Your IP

**DO NOT mention:**
- Specific LLM model performance metrics (unless very favorable)
- Exact recommendation algorithm implementation
- Proprietary pedagogy training data
- Specific school partnerships (until they go public)
- Exact feature roadmap (competitors could accelerate)
- Cost structure (unit economics, margin targets)
- Exact team names/contact info (until ready to hire)

**DO NOT share:**
- Closed source code
- Internal architecture diagrams (keep high-level)
- Financial model details (only summary numbers)
- Student data (even anonymized)
- Internal roadmap (stick to 4-phase public roadmap)

**INSTEAD:**
- Use the language from the pitch deck (tested, vetted)
- Speak in principles, not implementation details
- Focus on outcomes, not technical mechanics
- Emphasize moat without revealing moat details
