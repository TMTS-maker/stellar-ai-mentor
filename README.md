# Stellecta - AI-Powered Learning Platform

## Project Overview

Stellecta is a next-generation EdTech platform that combines AI-powered personalized learning with blockchain-verified credentials. Meet 8 specialized AI mentor agents, each with unique personalities and expertise.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd stellar-ai-mentor

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technology Stack

This project is built with:

- **Vite** - Build tool
- **TypeScript** - Type safety
- **React** - UI framework
- **shadcn-ui** - Component library
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations

## Build for Production

```sh
npm run build
```

## Project Structure

```
src/

├── components/      # React components
├── pages/          # Page components
├── data/           # Data and constants
├── hooks/          # Custom React hooks
└── lib/            # Utility functions
```

## Features

- 8 specialized AI mentor agents
- Interactive learning dashboard
- Real-time progress tracking
- Blockchain-verified credentials
- Parent, teacher, and student interfaces
- Gamification and rewards system
